# Backend of 'Roulette'

## Run

A way to run:

```bash
python3 -m .venv
. .venv/bin/activate
pip install -r requirements.txt
python3 src/app.py
```

Or (to resolve relative imports):

```bash
python3 main.py
```

Run in debug (auto-reload, etc.):
```bash
flask --app src/app.py run --debug
```

go to http://localhost:5000
